About the Platform

AutoClipper is a web tool designed to help creators automate the process of uploading and publishing short-form videos directly to TikTok using official developer APIs.

TikTok Integration

  • Login Kit — Allows users to securely authenticate with their TikTok account.
  • Content Posting API — Enables uploading and publishing videos directly to TikTok.

Permissions Requested

  • user.info.basic — Access basic profile information.
  • video.upload — Upload videos through the platform.
  • video.publish — Publish uploaded videos.
